# AegisX — Website for GitHub Pages

This repository contains the static website for AegisX.

## How to publish
1. Create a repository named `AegisX` under your GitHub account `ahmad-n00r`.
2. Push these files to the repository (root branch `main`).
3. In GitHub Settings → Pages, select **Branch: main** and **Folder: /(root)** and Save.
4. The site will be available at: `https://ahmad-n00r.github.io/AegisX/`.

## Files included
- index.html (Landing page)
- documentation.html
- downloads.html
- changelog.html
- assets/ (logo.svg, style.css, main.js)
